# Mealie

Recipe manager and meal planner. Import recipes from any URL, organize by category, plan meals, generate shopping lists.

## Setup

```bash
cd ~/services/mealie
docker compose up -d
# Open: http://localhost:9925
# Default login: changeme@example.com / MyPassword
```

## Port

| Port | Purpose |
|------|---------|
| 9925 | Web UI |
